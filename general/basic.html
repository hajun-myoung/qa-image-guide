기본 사용법

사이트 로그인

기본 로그인

위와 같은 로그인 페이지에서 아이디/비밀번호를 입력해주세요

회원가입 기능은 별도로 제공되고 있지 않습니다

모든 회원은 팀장님께 사전 보고후 아이디를 발급받으셔야 합니다

비밀번호 찾기

개인정보 보호를 위해, 비밀번호 찾기가 불가능합니다 (서버에도 비밀번호가 저장되지 않습니다)

비밀번호를 잊으신 경우, 팀장님께 연락하여 "비밀번호 초기화"를 부탁하세요

로그아웃

모든 탭을 종료했거나, 일정 시간 이상 움직임이 없다면 자동 로그아웃이 수행됩니다

다만, 시스템 설정에 따라 자동으로 로그아웃 되지 않을 수 있으므로,

사용이 끝나시면 우상단 로그아웃 버튼을 통해 로그아웃하시길 권장드립니다

교재 열람

기본 열람

교재는 위와 같은 "목록"에서 우선 접근이 가능합니다

준비된 썸네일들을 클릭해주세요

사이드바 사용

교재 열람 페이지에서는, 위와 같은 사이드바가 좌측에 제공됩니다

이 사이드바를 통해 교재 간 이동이 가능합니다

사이드바 상단의 공통 2026 과 같은 태그들은 "교재 분류"입니다
이 버튼을 클릭하면, 해당 카테고리에 있는 교재들을 보여줍니다
아무것도 선택되어있지 않다면, 전체 교재를 보여줍니다

인덱스 사이드바 사용

교재 열람 화면에서, 우측 화살표를 누르면 위와 같은 사이드바가 노출됩니다

이 사이드바를 통해 교재 페이지를 보다 편하게 찾으실 수 있습니다

페이지 이동

페이지 이동은 크게 두 가지 방법을 지원합니다:

  1. 위와 같이, 교재 열람 페이지의 화살표를 이용해 이전, 이후 페이지로 이동 가능합니다

  2. 열람 페이지 우하단에, 원하는 페이지 번호를 직접 입력하셔도 됩니다

확대 및 축소

열람 페이지 좌하단에, 확대 축소가 가능한 버튼이 준비되어 있습니다

기본 뷰어에서 두 번의 확대가 가능하며,
해당 버튼을 누를 때마다 조금씩 더 확대됩니다

두 번의 확대 후 해당 버튼을 다시 눌러주시면, 기본 보기로 축소됩니다

교재의 캡처 및 다운로드

교재의 캡처, PDF 파일 다운로드 등은 엄격하게 금지되고 있습니다

따라서 본 사이트에서 기본적으로 제공하지 않으며,
특정한 방법으로 위와 같은 작업(캡처, 다운로드)의 수행을 금합니다
