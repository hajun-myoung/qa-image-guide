# 교재 관리법

::: tip
본 메뉴에 접근 가능한 사용자는 `직원`급 이상입니다
:::

## 교재 파일관리 및 사이드바 관리

현재, 교재 파일관리 기능은 "사이드바 관리"와 그 기능이 통합되었습니다

일반적인 교재 생성의 순서는 다음과 같이 진행됩니다:

1. 사이드바에 해당 교재의 아이템을 만들어주세요

2. 이후, 만들어진 사이트바 아이템에 필요한 책을 업로드해주세요

## 교재 추가하기

![configureSidebar](/images/configureSidebar.png)

이 중, 교재 추가하기에 집중하여 관찰하면,

대분류, 카테고리 입력 칸의 하단에 `PDF 업로드` 버튼이 준비되었습니다

이를 통해, 새로운 교재 파일을 업로드할 수 있습니다

## 사이드바 수정하기

사이드바 수정은 크게 세 가지가 가능합니다

1. 위에서 이미 설명된 [교재 추가하기](#교재-추가하기)

1. 사이드바 수정하기

   이 기능(사이드바 수정)의 경우, 미리 구성된 사이드바의 교재 순서, 이름, 카테고리 및 태그 분류를 바꿀 수 있습니다

1. 인덱스 편집하기

   교재 내의 인덱스는 수동으로 입력하셔야 합니다 왼쪽 메뉴에서 "인덱스 편집하기"를 눌러주셔야 하며,

   인덱스의 쪽수가 해당 교재 전체 페이지 수보다 작거나 낙타야 합니다
