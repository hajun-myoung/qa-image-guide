사용자 관리법

팁

본 메뉴에 접근 가능한 사용자는 팀장급 이상입니다

홈페이지 - 우상단 설정 버튼을 통해 들어갈 수 있는 설정 페이지에는,
팀장급 이상의 계정으로 로그인했을 시에만 보이는 메뉴 사용자 관리가 있습니다

이 메뉴들로 사용자를 관리할 수 있습니다

사용자 목록

위와 같은 화면으로 기본적인 "사용자 목록"을 확인하실 수 있으며,

같은 화면에서 검색기를 사용해 특정 사용자를 검색할 수 있습니다.

검색 가능한 요소는 이름 또는 아이디입니다

사용자 생성

같은 표의 우상단, 새로 추가하기 버튼을 통해 새로운 사용자를 추가할 수 있습니다

이 때, 새로 추가할 사용자의 이름, 아이디, 비밀번호, 역할을 입력해주셔야 합니다

입력이 필요한 정보는 팝업(모달)창으로 뜨니, 알맞은 칸에 입력해주세요

비밀번호 초기화

간혹 비밀번호를 잊은 직원이 발생할 수 있습니다

보안 상의 이유로 잊은 비밀번호를 찾는 것은 불가능하지만, 기본 비밀번호로 초기화하는 기능이 준비되었습니다

위와 같이, 각 사용자 정보의 오른쪽 끝의 점 세개 버튼을 누르고,

따라오는 팝업 창에서 "비밀번호 초기화"를 눌러주시면 됩니다

사용자 비활성화

퇴사하거나, 사용을 중지해야하는 직원이 있다면 더 이상 사이트에 접근하지 못하도록 처리해주세요

비밀번호 초기화와 같은 단계를 거쳐, 마지막 팝업 창에서 비활성화를 선택해주시면 됩니다

사용자 "삭제"가 아닌 이유

복구의 가능성을 남겨두기 위해, 삭제가 아닌 비활성화가 채택되었습니다

복구 필요시 개발자에게 연락해주세요

로그인 기록

보안 상의 이유로 로그인 기록의 필요한 경우를 대비,

모든 사용자의 로그인 기록은 서버에 저장되고 있습니다.

필요 시, 이 부분에 대한 개인정보 제공 동의서를 드릴 수 있습니다

기록되는 정보들은 아래와 같습니다:

  • 사용자 이름(아이디)

  • 이 기록이 생성된 시각(로그인 시각)

  • 사용자별 고유식별번호(UID), 동명이인 등을 구분하기 위해 사용됩니다

  • 역할(직급)

  • IP 주소: 로그인한 당시 사용하고 있던 인터넷의 IP주소입니다

  • 사용중인 브라우저 정보: 크롬, 사파리 등의 브라우저 정보와 이를 구동중인 운영체제에 대한 정보를 기록합니다
